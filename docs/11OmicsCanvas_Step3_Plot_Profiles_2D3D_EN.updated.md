## 2. Input matrix naming (standard vs legacy)

Matrix file mapping rule:

`matrix_path = <matrix-dir>/<sample_prefix><suffix_by_gene_type>` ŅłĆfileciteŅłéturn6file0ŅłéL27-L34Ņłü

This script supports two filename conventions controlled by `--naming`.

### 2.1 Standard naming (recommended)

Default when `--naming standard`:

- **TSS** window: `*_tss_matrix.tsv`
- **gene profile**: `*_gene_profile_matrix.tsv`
- **TES** window: `*_tes_matrix.tsv` ŅłĆfileciteŅłéturn6file0ŅłéL12-L19Ņłü

### 2.2 Legacy naming

When `--naming legacy`:

- **TSS** window: `*_start_matrix.txt`
- **gene profile**: `*_gene_body_matrix.txt`
- **TES** window: `*_end_matrix.txt` ŅłĆfileciteŅłéturn6file0ŅłéL16-L20Ņłü

Advanced (usually unnecessary): you *can* override suffixes using hidden options:

- `--suffix-tss` / `--suffix-start`
- `--suffix-gene`
- `--suffix-tes` / `--suffix-end` ŅłĆfileciteŅłéturn6file0ŅłéL868-L871Ņłü

## 3. The key arguments (corrected to match the script)

### 3.1 Required

- `--mode {2d,3d}` ŅłĆfileciteŅłéturn6file0ŅłéL679-L691Ņłü

- `--matrix-dir DIR`  
  Directory containing matrix files (default: `caculate_matrix`) ŅłĆfileciteŅłéturn6file0ŅłéL693-L715Ņłü

  - `;` = panels stacked in the same column

- `--names STRING`  
  Legend names for each sample line. **Must match `--group` structure exactly** (same columns/panels/sample counts) ŅłĆfileciteŅłéturn6file0ŅłéL760-L767Ņłü ŅłĆfileciteŅłéturn6file0ŅłéL413-L426Ņłü

- `--out FILE`  
  Output figure path (`.pdf`, `.png`, `.svg`, ŌĆ”) ŅłĆfileciteŅłéturn6file0ŅłéL997-L1003Ņłü

> Convenience behavior:  
> If your `--group` has multiple columns but `--names/--ylabels` do **not** include `|`, the script will treat them as one continuous panel list and distribute sequentially across columns. The total number of panels must still match. ŅłĆfileciteŅłéturn6file0ŅłéL360-L412Ņłü

## 6. Troubleshooting

- **File not found**: check `--matrix-dir`, `--naming`, and your prefixes in `--group` (the script searches `<matrix-dir>/<prefix><suffix>`) ŅłĆfileciteŅłéturn6file0ŅłéL437-L448Ņłü
- **Structure mismatch**: `--names` must mirror `--group` exactly (per column/panel/sample count) ŅłĆfileciteŅłéturn6file0ŅłéL413-L426Ņłü
- **Empty plots**: often caused by `--index-filter` removing all genes; set `--index-filter ''` to disable ŅłĆfileciteŅłéturn6file0ŅłéL872-L889Ņłü
- **Layout overflow in 3D**: reduce `--col-width/--panel-height/--x-off/--y-off` or increase figure size; use `--strict-layout` to catch geometry issues early ŅłĆfileciteŅłéturn6file0ŅłéL564-L634Ņłü ŅłĆfileciteŅłéturn6file0ŅłéL1112-L1123Ņłü
